# Sample Images

This directory contains sample images for documentation purposes. These images showcase examples from each dataset used in the project.

## Required Images

Please add the following sample images to this directory:

### 1. `license_plate_sample.jpg`
- A sample image from the License Plate Detection Dataset
- Should show a Vietnamese vehicle with visible license plate
- Preferably with bounding box annotation visible
- Recommended size: 800x600 pixels or similar

### 2. `mini_car_sample.jpg`
- A sample image from the Mini Car Detect Dataset
- Should show toy/model cars
- Can include detection bounding box
- Recommended size: 640x640 pixels or similar

### 3. `real_car_sample.jpg`
- A sample image from the Real Car Detect Dataset
- Should show real vehicles in parking environment
- Can include detection annotations
- Recommended size: 640x640 pixels or similar

### 4. `veri_sample.jpg`
- A sample image from the VeRi Dataset
- Should show a vehicle from traffic surveillance camera
- Preferably showing clear vehicle features for re-identification
- Recommended size: Variable (maintain original aspect ratio)

## Usage

These images are referenced in the main [README.md](../../README.md) file to provide visual examples of the datasets.

## Image Guidelines

- Use representative images that showcase dataset characteristics
- Ensure images don't contain sensitive information
- Compress images appropriately for web viewing
- Maintain reasonable file sizes (< 500KB per image)
- Use JPEG format for photographs
